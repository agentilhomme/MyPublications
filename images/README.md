# Images

Drop image files here, then re-run `python -X utf8 scripts/build_page.py` —
it will pick them up automatically. Until then, the page shows placeholder
boxes.

- **Headshot:** `headshot.jpg` (or `.jpeg` / `.png` / `.webp`) — shown as a
  circular photo in the page header.
- **Fieldwork/research photos:** `fieldwork-1.jpg`, `fieldwork-2.jpg`,
  `fieldwork-3.jpg`, etc. (any `.jpg`/`.jpeg`/`.png`/`.webp` file whose name
  starts with `fieldwork`) — shown in the "Fieldwork & research" gallery
  near the bottom of the page, in filename order.
